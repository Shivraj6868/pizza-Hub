# 🍕 PizzaHub – Restaurant Website

## 🚀 Project Overview
PizzaHub is a modern and visually appealing restaurant landing page built using **HTML5 and CSS3**.  
It showcases a pizza restaurant with sections like Home, About, Menu, Signature Pizzas, Reviews, and Footer.

This project is built for practicing frontend development and UI design skills.
