# 🍕 PizzaHub - Restaurant Website

A modern and visually appealing pizza restaurant website built using **HTML5 and CSS3**.  
This project focuses on creating a premium restaurant landing page with attractive UI, responsive sections, and a clean dark-themed design.

## 📌 About The Project

PizzaHub is a restaurant website designed to showcase pizza varieties, restaurant information, and customer experience.

The main goal of this project was to practice frontend development concepts like:

- HTML structure
- CSS styling
- Layout designing
- Background images
- Navigation bar creation
- Section alignment
- Visual UI improvement
